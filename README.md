## 📊 Analyse de Campagne Marketing & Rétention Client
# 📝 Présentation du Projet
Ce projet analyse les données marketing d'une entreprise de services en ligne pour évaluer la performance de ses campagnes d'acquisition et la fidélité de ses clients. L'objectif est de transformer des données brutes en insights exploitables pour optimiser les budgets marketing.

# 🎯 Objectifs Business
Performance des Campagnes : Analyser le taux de conversion global et par canal.

Segmentation Client : Identifier les canaux les plus rentables par tranche d'âge et par langue.

Analyse de la Rétention : Mesurer la qualité de l'acquisition en calculant le taux de rétention des nouveaux abonnés.

Optimisation Temporelle : Déterminer les jours de la semaine les plus performants pour diffuser les publicités.

# 🛠️ Compétences Techniques Démontrées
Nettoyage & Manipulation de Données : Renommage de variables, typage des données (dates, booléens), gestion des valeurs manquantes.

Feature Engineering : Création de nouvelles variables (langue appropriée, jour de semaine, encodage des canaux).

Analyse Statistique : Création de fonctions personnalisées pour automatiser le calcul des taux de conversion sur plusieurs segments.

Visualisation de Données : Utilisation de Matplotlib et Seaborn pour illustrer les tendances temporelles et les performances comparées.

# 📈 Résultats Clés
Taux de Conversion Global : 13.89 %.

Top Canal d'Acquisition : L'Email présente le meilleur taux de rétention (87.58 %) et une performance de conversion élevée.

Volume d'utilisateurs : Analyse effectuée sur une base de 7 309 utilisateurs uniques.

# 📂 Structure des Données
Le dataset comprend 12 colonnes clés, notamment :

id_utilisateur : Identifiant unique du client.

canal_marketing : Canal d'exposition (Facebook, Instagram, Email, etc.).

converti : Indicateur de succès de la campagne (True/False).

retention_active : Indique si le client est toujours abonné après sa période d'essai.

Conclusion
Ce projet démontre comment l'utilisation de la librairie Pandas permet d'extraire des indicateurs clés de performance (KPIs) essentiels pour un département marketing stratégique.
